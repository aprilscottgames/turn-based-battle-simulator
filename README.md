# Turn-Based Battle Simulator (C#)

A turn-based combat simulator built in C# demonstrating core gameplay systems, player actions, enemy AI, and game state management.

## Features
- Player actions: Attack, Heal, Defend
- Enemy AI with randomized behavior
- Turn-based system with state management
- Health tracking and win/lose conditions

## Technologies
- C#
- .NET Console Application

## How to Run
```bash
dotnet run
```
## Purpose
Built to practice gameplay programming concepts in C#, including turn systems, state management, and core game logic for game development roles.
